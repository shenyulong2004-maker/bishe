## 登录页 UI 上线检查清单

### 兼容性
- 桌面端：Chrome/Edge 最新版
- 移动端：Android Chrome / iOS Safari（重点检查键盘弹出后布局不抖动、可滚动）
- 深色模式：系统深色模式下颜色与对比度正常

### 无障碍（WCAG 2.1）
- Tab 可按顺序聚焦到所有可操作控件
- 聚焦态清晰可见（输入框与按钮都有明显 focus ring）
- 错误提示在界面可见且可理解（空提交/缺字段）
- 浏览器缩放 200%：布局不破坏、仍可完成登录

### 功能回归
- 账号/密码输入与 Enter 提交
- 多角色时用户类型选择
- 记住密码（学生端）开关可用
- 登录失败提示不遮挡输入（仍可继续修正）

### 性能与稳定性
- 首屏加载无明显卡顿（背景图/Canvas 不阻塞交互）
- 登录按钮 loading 防重复提交

### 配置与部署
- 前端环境变量（VUE_APP_BASE_API 等）与后端一致
- 静态资源路径正确（背景图可访问）
- 生产构建产物检查：hash 资源正常、缓存策略符合预期

### 自动化测试
- Playwright 用例全部通过
- 如端口不同，确认 PLAYWRIGHT_BASE_URL 设置正确
